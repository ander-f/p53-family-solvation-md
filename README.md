# p53-family-solvation-md

This repository contains **solvation analysis data**, **initial and final coordinates**, 
and **scripts** for microsecond-long molecular dynamics simulations of the p53 family proteins (p53, p63, and p73).

## Directory structure
- `analyses/`: solvation analysis results
- `coordinates/`: initial and final PDB coordinate files
- `mdp_files/`:  mdp files for setting up simulations
- `scripts/`: analysis scripts
